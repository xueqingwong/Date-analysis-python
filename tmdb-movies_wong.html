TMDb电影数据分析¶

研究目的：本文通过对电影数据库TMDb的1万条从1968～2015年电影数据的清理和分析，探讨二十世纪以来电影产业的发展规律，
并进一步探究何种特征的电影会对票房收益和用户评价产生积极影响，以期为电影产业的发展留下一些思考和结论。

分析思路：

一，数据数据读取/评估和清理 1，数据结构 2，异常值处理

二，数据分析
PART 1：电影产业规律分析

1，票房收益角度，选取字段调整后的收入(revenue_adj) - 调整后的预算(budget_adj)，得到调整后的收益(profit_adj)；
2，用户评价角度，选取查看次数popularity和平均评分vote_average字段；

回答以下问题：

Q1，整体电影产业收益是否在增长，年总票房盈利的变化？
Q2，各电影公司每年盈利是否在增长，年均票房盈利的变化？
Q3，大众对电影的关注是否呈现逐年增加的趋势？
Q4，随着电影技术的发展，当前发行电影的评分是否也在增加？
Q5，占领市场最多的影片类型/电影公司/导演和演员？

PART 2：评价角度的选取

1，从票房收益和用户评价两个角度来进行电影评价；
2，考虑选取字段自身的影响因素和相互间的关系，从而确定评价电影的因变量；

回答以下问题

Q1，相对网页查看次数popularity和平均评分vote_average之间的相关性，是否查看多的影片评分就高？
Q2，分析用户评价（包括查看次数和平均评分）和票房收益之间的相关性，从而确定评价电影的指标为何？

PART 3：影响电影收益和评价的因素分析

考虑什么特征的电影会票房收益高/用户评价好，回答以下问题：

Q1，电影时长runtime分析，主要时长是多少分钟，是否对票房收益和电影评价有影响？
Q2，电影发行月份分析，主要发行月份集中在哪个月，是否对票房收益和电影评价有影响？

三，结论
1，研究问题总结
1）20世纪电影产业发展规律
2）电影收益和评价影响因素及策略

2，不足与展望

def extract_data(column_name):
    all_data = df[column_name].str.cat(sep = '|')
    all_data = pd.Series(all_data.split('|'))
    count = all_data.value_counts(ascending = False)    
    return count

三，结论¶

1，研究问题总结¶

# 20世纪电影产业发展规律

1）20世纪以来，电影产业的票房年收益虽然整体是增长的，但主要原因是每年发行的电影数量变多，平均到每部片的年均盈利实际是变少的，整个行业的竞争在加剧；
2）20世纪以来，大众对电影的关注度在持续增加，但同步观众的期望值也越来越高，评分要求越来越严格，随着年份的增长，观众对当前发行电影的评分是下降的，市场要求越来越高质的电影；
3）当前电影市场占有率集中于某些类型的电影/电影公司/导演及演员上，出现频率最高的电影类型为戏剧，电影公司是环球影业，导演是伍迪·艾伦，演员是罗伯特·德尼罗。

# 电影收益和评价影响因素及策略

1）相对网页查看次数popularity此因素与电影的票房收益和观众评价都有强正相关的关系，电影公司可以注意在上映前在互联网途径上新影片的介绍和宣传；
2）时长100～200分钟的电影票房收益较高，电影的拍摄最好选择此时长区间；
3）电影发布月份偏好9和10两个月，但实际选择6月和12月发布的电影票房收益较高，电影公司可以选择6月或12月这两个月份的蓝海进行开拓。

2，不足与展望¶

• 所探索的特征仍是有限的，后续研究者可继续在电影公司/导演/演员因素考虑年份深入探究对票房收益及影评的影响；
• 去掉所有异常值后分析的样本只有3747个，样本的骤减可能会对分析的准确度造成一定影响；
• 本项目只探索了特征之间的相互关系，只是分析高票房和影片的影片它们有的一些相关特征，并没有他们之间的因果关系。
